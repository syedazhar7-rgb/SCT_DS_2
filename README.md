# 🚢 Titanic Dataset - Data Cleaning & Exploratory Data Analysis (EDA)

## 📌 Internship Task
This project is part of **Task 02** of the **SkillCraft Technology Data Science Internship**.

The objective of this task is to perform **Data Cleaning and Exploratory Data Analysis (EDA)** on a dataset to discover patterns, relationships, and insights from the data.

For this project, the **Titanic Dataset from Kaggle** is used.


## 📊 Project Objective
The main goals of this project are:

- Perform **data cleaning**
- Handle **missing values**
- Understand the **structure of the dataset**
- Perform **exploratory data analysis**
- Identify **patterns and relationships** between variables
- Visualize important insights using graphs

## 🗂 Dataset
Dataset used: **Titanic Dataset**

It contains information about passengers such as:
- Passenger Class
- Age
- Gender
- Fare
- Cabin
- Embarked
- Survival Status

Dataset Source: Kaggle Titanic Dataset


## 🛠 Tools & Libraries Used

- **Python**
- **Pandas** – Data manipulation
- **Matplotlib** – Data visualization
- **Seaborn** – Statistical visualization
- **Jupyter Notebook** – Analysis environment


## 🔎 Exploratory Data Analysis

Some of the analysis performed in this project includes:

- Survival count visualization
- Survival analysis based on **Gender**
- Survival analysis based on **Passenger Class**
- Age distribution of passengers
- Relationship between **Fare and Survival**
- Correlation analysis between numerical variables


## 📈 Key Insights

- **Females had a higher survival rate compared to males**
- **Passengers in higher classes (Pclass 1)** had better survival chances
- **Younger passengers** had slightly higher survival probability
- Ticket fare also showed some relationship with survival rate
